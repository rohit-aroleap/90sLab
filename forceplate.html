The grid above reflects each cell's calibration state from [STATUS]. Calibrate all cells sends c to the device, which starts the 2-point calibration wizard. The wizard is interactive and currently runs on the USB serial port only — open pio device monitor (115200 baud) on a computer attached to the ESP32 and answer the prompts there. Once calibration finishes, the updated scale factors appear here automatically. Reset all calibration (x) works over WebSocket.

WebSocket Protocol

Default: ws://force-plate.local:80
Posting: [<ms>ms] FL:<g>g FR:<g>g BL:<g>g BR:<g>g TOTAL:<g>g
Status: [STATUS] {json}
Cal: [CAL] / [CAL:<name>] prompts
Info: [INFO] message
Commands: h · s · d · l · r · start · read · stop · t[1-4] · c · x[1-4]

CSV format

timestamp, elapsed_s, device_ms, fl_g, fr_g, bl_g, br_g, total_g, cop_x_mm, cop_y_mm

Signal processing

COP computed from the 4 calibrated corner forces (plate 339.411 × 339.411 mm). 2nd-order Butterworth low-pass at 5 Hz cutoff (sample rate 40 Hz). Metrics (sway RMS, path length, mean velocity, 95 % confidence ellipse area, AP/ML range, max distance, composite balance score) recomputed every 8 samples over a rolling 800-sample (10 s) window. 500 ms warmup at session start discards step-on artefact.
